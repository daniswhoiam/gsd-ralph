- Resolve SCRIPT_DIR: `SCRIPT_DIR="$(cd "$BATS_TEST_DIRNAME/../src" && pwd)"`

- `rm -rf "$TEST_DIR"`

4 test cases:

2. `@test "add stores correct description"` -- run cmd_add "Buy milk", verify `jq -r '.[0].description' "$TASKCTL_DATA"` equals "Buy milk"
3. `@test "add multiple tasks increments IDs"` -- add two tasks, verify first has id 1, second has id 2
4. `@test "add with special characters"` -- add "Task with 'quotes' & symbols", assert_success, verify description stored correctly

Use `run cmd_add ...` pattern with bats-assert's `assert_success` and `assert_output --partial` where appropriate.
