    **GREEN phase:** Create install.sh at the repo root implementing:

    1. **Shebang and strict mode:** `#!/usr/bin/env bash` with `set -euo pipefail`

    2. **Self-resolution:** Copy the proven BASH_SOURCE symlink resolution pattern from bin/gsd-ralph (see research Pattern 1). Since install.sh is at repo root, `GSD_RALPH_REPO` equals the resolved script directory directly.

    3. **Color output:** Copy the terminal-detection color pattern from lib/common.sh (see research Code Examples). Define print_success, print_warning, print_error, print_info functions.

    4. **check_prerequisites():** Check in order:
       - bash version >= 3.2 via BASH_VERSINFO (see research Pattern 4)
       - `command -v git` with install URL on failure
       - `command -v jq` with brew/apt instructions on failure
       - `.planning/` directory exists (GSD framework check)
       - `.planning/config.json` exists
       - Self-install guard: `[[ "$(cd "$GSD_RALPH_REPO" && pwd)" == "$(pwd)" ]]` should fail with error
       Return non-zero if any check fails. Print ALL failures before returning (don't exit on first).

    5. **install_file():** Implement the idempotent file copy function from research Pattern 2. Takes source, destination, and executable flag. Uses `cmp -s` for comparison. Tracks INSTALLED and SKIPPED counters.

    6. **install_command_file():** Special handler for ralph.md that uses sed to adjust the path from `scripts/ralph-launcher.sh` to `scripts/ralph/ralph-launcher.sh` (research Pattern 5). For idempotency comparison, generate the sed-modified version to a temp file first, then cmp against existing target.

    7. **main flow:**
       ```
       check_prerequisites || exit 1
       INSTALLED=0 SKIPPED=0
       install_file "$GSD_RALPH_REPO/scripts/ralph-launcher.sh" "scripts/ralph/ralph-launcher.sh" true
       install_file "$GSD_RALPH_REPO/scripts/assemble-context.sh" "scripts/ralph/assemble-context.sh" true

       ```
