       `_format_duration()` -- Args: $1=seconds. Computes mins and secs via integer arithmetic. Prints "Nm Ns" if mins > 0, else "Ns".

       `_init_audit_log()` -- Args: $1=audit_file. Creates parent directory, truncates file with `: > "$audit_file"`.

       `_print_audit_summary()` -- Args: $1=audit_file. If file exists and is non-empty (`-s`), counts lines with `wc -l`, prints "Audit: N decisions logged. See $audit_file for details."

         # Check circuit breaker (wall-clock timeout)
         if ! _check_circuit_breaker "$TIMEOUT_MINUTES" "$loop_start_epoch"; then

         # Check graceful stop
