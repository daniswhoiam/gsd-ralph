**Step B: Create bench-eval.sh** under `benchmarks/harness/`.

```bash
#!/bin/bash
set -euo pipefail
CHALLENGE="${1:?Usage: bench-eval.sh <challenge-name> [taskctl-dir]}"
TASKCTL_DIR="${2:-benchmarks/taskctl}"
```

The script must:
1. Resolve its own directory and the challenges directory using `BASH_SOURCE[0]`
2. Find and validate the challenge JSON file (`$CHALLENGES_DIR/$CHALLENGE.json`)
3. Extract `check_script` and `name` fields using jq
4. Print a header: `=== Evaluating: $CHALLENGE_NAME ===`
5. Run the check script: `bash "$CHALLENGES_DIR/$CHECK_SCRIPT" "$TASKCTL_DIR"`
6. Capture exit code
7. Print `RESULT: PASS` or `RESULT: FAIL` based on exit code
8. If challenge JSON not found, print available challenges and exit 1

Mark bench-eval.sh as executable.
